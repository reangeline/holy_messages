# Backgrounds para Versículos Premium

## Instruções
Adicione aqui as imagens de fundo que os usuários premium podem usar ao compartilhar versículos.

## Formato recomendado:
- **Formato**: PNG ou JPG
- **Tamanho**: 1080x1920 (formato vertical de story) ou 1200x1200 (quadrado)
- **Nomenclatura**: Use nomes descritivos (ex: `sunset.jpg`, `mountains.png`, `nature1.jpg`)

## Exemplos de temas:
- Natureza (céu, montanhas, mar)
- Abstratos (gradientes, texturas)
- Espirituais (cruzes, igreja, luz)
- Minimalistas (cores sólidas, suaves)

As imagens devem ter boa legibilidade para texto branco ou preto sobreposto.
